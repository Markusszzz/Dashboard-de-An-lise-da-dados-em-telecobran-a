# Dashboard-de-An-lise-da-dados-em-telecobran-a
Projeto de Dashboard de Cobrança desenvolvido com Power BI, SQL e análise de dados, focado em indicadores de inadimplência, recuperação financeira e performance operacional. O dashboard apresenta KPIs, gráficos interativos e insights estratégicos para apoio na tomada de decisão baseada em dados.
